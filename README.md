# learn-expressjs

## To run this App

### Clone App

```shell
$ git clone https://github.com/kumarmanish03/learn-expressjs.git
```

### Download Dependencies and Modules

```shell
$ git npm init
```

### Run App

```shell
$ git npm run dev
```

Console must shows 

'''javascript
  Server running on PORT 5000...
 '''
 
 ## Go to the browser and type
 
 '''javascript
 http://localhost:5000/
 ''' 
 
 ** Enjoy! Your app is running  :)
